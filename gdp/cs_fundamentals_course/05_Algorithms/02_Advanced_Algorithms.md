# Advanced Algorithms

This is a list of more complex algorithms for advanced learners.

### Sorting Algorithms

1.  **Merge Sort:** A very efficient and stable sorting algorithm. It's a great introduction to the "divide and conquer" strategy.
2.  **Quick Sort:** Generally faster than Merge Sort in practice, but not stable. Also a "divide and conquer" algorithm.
3.  **Heap Sort:** An in-place sorting algorithm with worst-case O(n log n) runtime.
4.  **Radix Sort:** A non-comparative sorting algorithm that works in linear time.

### Graph Algorithms

1.  **Breadth-First Search (BFS):** A fundamental graph traversal algorithm.
2.  **Depth-First Search (DFS):** Another fundamental graph traversal algorithm.
3.  **Dijkstra's Algorithm:** An algorithm for finding the shortest paths between nodes in a graph.
4.  **Bellman-Ford Algorithm:** An algorithm that computes shortest paths from a single source vertex to all of the other vertices in a weighted digraph.
5.  **Floyd-Warshall Algorithm:** An algorithm for finding shortest paths in a weighted graph with positive or negative edge weights.

### Dynamic Programming

1.  **Fibonacci Sequence:** A classic dynamic programming problem.
2.  **Knapsack Problem:** A problem in combinatorial optimization.
3.  **Longest Common Subsequence:** A problem of finding the longest subsequence common to all sequences in a set of sequences.

### String Matching Algorithms

1.  **Naive String Matching:** The simplest method to solve the string matching problem.
2.  **Knuth-Morris-Pratt (KMP) Algorithm:** A more efficient string-searching algorithm.

### Greedy Algorithms

1.  **Activity Selection Problem:** A combinatorial optimization problem.
2.  **Huffman Coding:** A particular type of optimal prefix code that is commonly used for lossless data compression.

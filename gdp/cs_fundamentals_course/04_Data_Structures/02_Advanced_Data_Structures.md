# Advanced Data Structures

This is a list of more complex data structures for advanced learners.

1.  **Hash Tables:** A data structure that implements an associative array abstract data type, a structure that can map keys to values. It is a very powerful and widely used data structure.
2.  **Trees:** A hierarchical data structure.
    *   Binary Tree
    *   Binary Search Tree (BST)
    *   AVL Tree
    *   Red-Black Tree
3.  **Heaps:** A specialized tree-based data structure that satisfies the heap property.
    *   Min-Heap
    *   Max-Heap
4.  **Tries:** A tree-like data structure that proves to be very efficient for solving problems related to strings.
5.  **Graphs:** A non-linear data structure consisting of nodes and edges.
    *   Directed Graph
    *   Undirected Graph
    *   Weighted Graph

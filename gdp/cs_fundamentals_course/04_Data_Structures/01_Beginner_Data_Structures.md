# Beginner Data Structures

This is a list of fundamental data structures ideal for beginners.

1.  **Arrays:** A collection of items stored at contiguous memory locations. This is the most basic data structure.
2.  **Strings:** A sequence of characters, which can be thought of as an array of characters.
3.  **Linked Lists:** A linear collection of data elements where the order is not given by their physical placement in memory. They are a great introduction to pointers and memory management.
    *   Singly Linked List
    *   Doubly Linked List
    *   Circular Linked List
4.  **Stacks:** A LIFO (Last-In, First-Out) data structure. It's conceptually simple and used in many algorithms.
5.  **Queues:** A FIFO (First-In, First-Out) data structure. Also conceptually simple and widely used.
